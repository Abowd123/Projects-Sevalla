# 🎛️ تخصيص ردود البوت (للمطوّر فقط)

تقدر تغيّر نص أي رد يرسله البوت، ويُطبّق التغيير على **كل القروبات** تلقائياً.
الأوامر تعمل داخل القروبات وفي **خاص البوت**، وهي مخصّصة **للمطوّر/المالك فقط**.

## تغيير رد
1. ردّ على رسالة البوت التي تريد تغييرها واكتب: `تغيير`
2. البوت يطلب النص الجديد — أرسله (أو اكتب `الغاء` للإلغاء).

### المتغيّرات داخل النص
بعض الردود تحتوي متغيّرات (اسم العضو، مبلغ، عدد...). في النص الجديد تُكتب كأرقام بين قوسين:
- `{0}` = المتغيّر الأول (غالباً اسم/معرّف العضو)
- `{1}`، `{2}` = المتغيّرات التالية بالترتيب

مثال: الرد الأصلي «ماعندك فلوس ارسل الالعاب» تكتبه:
```
يا {0} رصيدك صفر! العب عشان تجمع فلوس
```
فيظهر لكل عضو باسمه تلقائياً. (المتغيّرات اختيارية — لو ماكتبتها يظهر النص بدونها.)

## استرجاع رد واحد
ردّ على رسالة البوت واكتب: `استرجاع الرد`  → يعيد الرد الأصلي.

## عرض الردود المعدّلة
اكتب: `الردود المعدلة`  (أو `قائمة الردود`) → يعرض قائمة (القديم ⇜ الجديد).

## استرجاع كل الردود
اكتب: `استرجاع كل الردود`  (أو `تصفير الردود`) → يعيد جميع الردود للأصل دفعة واحدة.

## ملاحظات
- الردود المتطابقة نصّاً تتشارك تخصيصاً واحداً في كل الملفات (تغييرها مرة يطبّقها أينما ظهرت).
- غير مشمول: النوافذ المنبثقة للأزرار (answer)، والردود المبنية في متغيّر ثم تُرسل.
